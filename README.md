## О проекте
 
Fox Defender — это промо-страница для будущей VR-игры. Игровая идея: пользователь становится последним рубежом обороны в небе и защищает самолёт от пушистых захватчиков.
 
На сайте представлены:
 
- описание сюжета и геймплея;
- список ключевых особенностей игры;
- блок о VR и Unity-технологиях;
- дорожная карта разработки;
- ссылка на поддержку проекта через DonationAlerts.
 
## Возможности лендинга
 
- адаптивная навигация с мобильным меню;
- hero-блок с фоновым изображением и CTA-кнопками;
- анимированные карточки особенностей;
- canvas-частицы на фоне;
- эффект свечения курсора;
- плавная прокрутка к секциям;
- roadmap разработки;
- локальный счётчик поддержавших в `localStorage`.
 
## Технологии
 
Проект сделан как статический сайт без сборщика:
 
- HTML5;
- CSS3;
- Vanilla JavaScript;
- Google Fonts;
- SVG/PNG-ассеты.
 
Для запуска не нужны Node.js, npm, Vite, Webpack или другие инструменты сборки.

## Быстрый старт
 
1. Клонируйте репозиторий:
 
   ```bash
   git clone https://github.com/MedBet/fox-defender.git
   cd fox-defender
   ```
 
2. Запустите локальный статический сервер:
 
   ```bash
   python3 -m http.server 8000
   ```
 
3. Откройте сайт в браузере:
 
   ```text
   http://localhost:8000
   ```
 
Также можно открыть `index.html` напрямую в браузере, но локальный сервер удобнее для проверки путей к ассетам и поведения страницы.
 
## Разработка
 
Основные файлы для изменения:
 
- `index.html` — структура секций, тексты, ссылки;
- `styles.css` — визуальный стиль, сетки, анимации, адаптив;
- `script.js` — частицы, меню, плавная прокрутка, интерактивные эффекты.
 
### Проверка после изменений
 
После правок рекомендуется:
 
1. Запустить локальный сервер:
 
   ```bash
   python3 -m http.server 8000
   ```
 
2. Проверить страницу в браузере на desktop и mobile-разрешениях.
3. Убедиться, что:
   - меню открывается на мобильной ширине;
   - ссылки навигации плавно ведут к нужным секциям;
   - изображения и иконки загружаются;
   - кнопка поддержки открывает DonationAlerts;
   - страница не показывает ошибок в консоли браузера.
